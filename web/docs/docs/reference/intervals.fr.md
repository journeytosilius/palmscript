# Table Des Intervalles

PalmScript prend en charge tous les intervalles actuels de kline Binance.

| Literal | Meaning |
| --- | --- |
| `1s` | 1 seconde |
| `1m` | 1 minute |
| `3m` | 3 minutes |
| `5m` | 5 minutes |
| `15m` | 15 minutes |
| `30m` | 30 minutes |
| `1h` | 1 heure |
| `2h` | 2 heures |
| `4h` | 4 heures |
| `6h` | 6 heures |
| `8h` | 8 heures |
| `12h` | 12 heures |
| `1d` | 1 jour |
| `3d` | 3 jours |
| `1w` | 1 semaine |
| `1M` | 1 mois |

Rappels de casse :

- hebdomadaire est `1w`
- mensuel est `1M`
