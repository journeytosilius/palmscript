# Glossario

## Intervalo Base

O intervalo de execucao declarado por `interval <...>`.

## Intervalo Declarado

Um intervalo introduzido explicitamente via `use <alias> <...>` para uma fonte
nomeada.

## Script Source-Aware

Um script que declara pelo menos uma `source`.

## Template De Source

Um construtor integrado de exchange / venue como `binance.spot` ou
`hyperliquid.perps`.

## Modo Mercado

Execucao contra feeds historicos ligados a mercado.

## Sem Lookahead

A garantia de que uma amostra de intervalo superior so fica visivel depois que
o candle fecha completamente.

## Serie De Saida

Um resultado nomeado por passo emitido por `export` ou `trigger`.

## Uniao Dos Timestamps Base

A timeline de execucao construida a partir da uniao dos tempos de abertura dos
candles de intervalo base de todas as fontes declaradas.
