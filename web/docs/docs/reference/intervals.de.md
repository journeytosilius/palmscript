# Intervalltabelle

PalmScript unterstuetzt alle aktuellen Binance-Kline-Intervalle.

| Literal | Bedeutung |
| --- | --- |
| `1s` | 1 Sekunde |
| `1m` | 1 Minute |
| `3m` | 3 Minuten |
| `5m` | 5 Minuten |
| `15m` | 15 Minuten |
| `30m` | 30 Minuten |
| `1h` | 1 Stunde |
| `2h` | 2 Stunden |
| `4h` | 4 Stunden |
| `6h` | 6 Stunden |
| `8h` | 8 Stunden |
| `12h` | 12 Stunden |
| `1d` | 1 Tag |
| `3d` | 3 Tage |
| `1w` | 1 Woche |
| `1M` | 1 Monat |

Hinweise zur Gross-/Kleinschreibung:

- woechentlich ist `1w`
- monatlich ist `1M`
