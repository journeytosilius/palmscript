# Tabla De Intervalos

PalmScript soporta todos los intervalos actuales de klines de Binance.

| Literal | Significado |
| --- | --- |
| `1s` | 1 segundo |
| `1m` | 1 minuto |
| `3m` | 3 minutos |
| `5m` | 5 minutos |
| `15m` | 15 minutos |
| `30m` | 30 minutos |
| `1h` | 1 hora |
| `2h` | 2 horas |
| `4h` | 4 horas |
| `6h` | 6 horas |
| `8h` | 8 horas |
| `12h` | 12 horas |
| `1d` | 1 dia |
| `3d` | 3 dias |
| `1w` | 1 semana |
| `1M` | 1 mes |

Recordatorios sensibles a mayusculas:

- semanal es `1w`
- mensual es `1M`
