# Installation

PalmScript wird offentlich derzeit uber die gehostete Browser-IDE demonstriert:

[https://palmscript.dev/](https://palmscript.dev/)

Bevor du den Beispielen in `Lernen` folgst, stelle sicher, dass:

- du die gehostete App erreichen kannst
- dein Browser JavaScript und WebAssembly erlaubt
- du Skripte in der IDE bearbeiten und ausfuhren kannst

Die offentlichen `Lernen`-Beispiele gehen davon aus, dass du PalmScript in der
Browser-IDE schreibst, dort die Diagnosen ansiehst und den BTCUSDT-Backtest-
Ablauf aus der App startest.
